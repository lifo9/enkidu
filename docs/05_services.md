# Services
1. Change dir to [services folder](../services).
2. Pick a service and change to its directory.
   ```shell
   cd <service>
   ```
3. Make sure all ENV variables are loaded.
4. Run Ansible playbook.
   ```shell
   ansible-playbook -i inventory.yaml site.yaml
   ```
