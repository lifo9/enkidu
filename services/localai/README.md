# Download models
Inside container, run:
```bash
wget https://huggingface.co/skeskinen/ggml/resolve/main/all-MiniLM-L12-v2/ggml-model-f32.bin -O /models/ggml-model-f32.bin

wget https://huggingface.co/TheBloke/dolphin-2.6-mistral-7B-GGUF/resolve/main/dolphin-2.6-mistral-7b.Q5_K_M.gguf -O /models/dolphin-2.6-mistral-7b.Q5_K_M.gguf

wget https://huggingface.co/TheBloke/dolphin-2.7-mixtral-8x7b-GGUF/resolve/main/dolphin-2.7-mixtral-8x7b.Q5_0.gguf -O /models/dolphin-2.7-mixtral-8x7b.Q5_0.gguf

wget https://huggingface.co/TheBloke/dolphin-2_2-yi-34b-GGUF/resolve/main/dolphin-2_2-yi-34b.Q4_K_M.gguf -O /models/dolphin-2_2-yi-34b.Q4_K_M.gguf
```
